# Adventure Works Excel Data Preparation

## Project Overview
This project is part of my data analytics learning journey. The task involved cleaning, structuring, and formatting financial data in Microsoft Excel to prepare it for presentation and analysis.

The dataset was provided by an Adventure Works executive and required verification, correction of data issues, and improved formatting for readability.

## Tools Used
- Microsoft Excel
- Data Cleaning Techniques
- Spreadsheet Formatting

## Objectives
- Verify and correct existing data in the worksheet
- Improve readability and formatting
- Add country information for international data
- Apply currency formatting for multiple currencies
- Import exchange rate information from an external document
- Organize and manage worksheets for presentation

## Tasks Completed

### 1. Data Cleaning
- Removed unnecessary columns
- Corrected numeric formatting issues
- Fixed data incorrectly stored as text
- Adjusted column widths for better visibility

### 2. Data Structuring
- Added a new **Country** column
- Populated country values using Autofill
- Inserted headings for different countries to improve clarity

### 3. Data Formatting
- Applied consistent formatting to column headers
- Adjusted column widths and enabled text wrapping
- Applied currency formatting for:
  - US Dollar
  - Euro
  - Japanese Yen
- Used Format Painter to maintain consistent styling

### 4. Adding External Data
- Created a new worksheet
- Imported exchange rate data from a PDF file
- Structured the data with the following fields:
  - USD Amount
  - Currency
  - Rate

### 5. Worksheet Management
- Renamed worksheets to meaningful names:
  - `Sample Figures`
  - `Exchange Rates`
- Organized worksheet order
- Hid the `Contacts` sheet for presentation purposes

## Key Skills Practiced
- Data cleaning
- Spreadsheet structuring
- Data formatting
- Working with multiple currencies
- Worksheet organization
- Preparing data for business presentations
