# Adventure Works Inventory Management

A Microsoft Excel project demonstrating data organization and sorting techniques for inventory management.  

This project focuses on improving readability and accessibility of a product inventory file, allowing management to quickly locate and analyze information.

---

## Project Overview

The Adventure Works Inventory.xlsx file contains one worksheet, **Products**, with **151 rows** and **9 columns**. The goal of this project was to reorganize the data to make it easier to read, track, and analyze for management review.

---

## Objectives

- Add visual markers to track row positions during sorting
- Perform alpha-numeric and multi-level sorts
- Improve efficiency when working with large datasets

---

## Tasks Completed

### 1. Adding a Visual Marker
- Selected **row 32** (columns A to I)  
- Applied a **yellow fill color** using the Home → Font → Fill Color tool  
- Purpose: Track row positions before and after sorting

### 2. Sorting the Data
- **Product Name (Ascending & Descending)**  
  - Sorted alphabetically to organize products for quick reference  

- **Date Entered (Oldest to Newest)**  
  - Sorted by date to monitor chronological inventory entries  

- **Supplier (Ascending)**  
  - Sorted suppliers alphabetically to organize stock sources  

- **Multi-level Sort: Supplier & Units in Stock**  
  - Sorted by **Supplier (A-Z)** and **Units in Stock (Largest to Smallest)**  
  - Reversed the sort using **Undo (Ctrl + Z)** to demonstrate workflow flexibility  

---

## Key Learnings

- Using **visual markers** helps verify sorting operations  
- Multi-level sorting allows complex organization of inventory data  
- Excel's sorting features improve efficiency and readability in large datasets  

---
