# Adventure Works Inventory Analysis – Filtering Exercise

This project demonstrates the use of Microsoft Excel **filtering techniques** to analyze an inventory dataset and extract key business insights.

---

## Project Overview

The Adventure Works Inventory1.xlsx file contains a **Products worksheet** with 151 rows and 9 columns. The objective of this exercise was to extract specific information for management review by applying filters.

---

## Tasks Completed

### 1. Turned on Filtering
- Enabled filters in the **Data → Filter** menu  
- Verified that filter drop-downs appeared for all columns

### 2. Extracted Specific Information Using Filters
- **Gear Components**: Applied a filter on the Category column to view only gear components  
- **Orders for Z123**: Filtered the Supplier column for Z123  
  - Note: The data did not include orders for 2023, but the filtering technique was demonstrated  
- **Mountain Bike Frames**: Filtered Product Name using **Text Filter → Contains**  
- **High Stock Mountain Bike Orders**: Applied a **Number Filter → Greater Than** on Units in Stock  

---

## Key Learnings

- Filtering allows quick analysis of large datasets  
- Text and number filters help extract precise information for management decisions  
- The exercise demonstrates skills in **data analysis, Excel navigation, and filtering logic**
