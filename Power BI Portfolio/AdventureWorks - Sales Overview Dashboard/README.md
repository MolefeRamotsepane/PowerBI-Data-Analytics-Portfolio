# AdventureWorks Sales Performance Dashboard

## Project Overview
This project analyzes sales performance using Power BI. The goal was to identify key revenue drivers, regional performance, and product trends.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Dataset
AdventureWorks Sales Dataset

## Key Business Insights

1. Bikes generate the majority of company revenue.
2. North America is the strongest performing region.
3. Monthly sales remain relatively stable throughout the period.
4. Hitch Bike – 4 Bike is the highest revenue generating product.
5. The company maintains a profit margin of approximately 88%, indicating strong profitability.

## Dashboard Features
- Sales KPI overview
- Product performance analysis
- Regional sales comparison
- Monthly revenue trends
